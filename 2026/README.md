# Procesarea și Vizualizarea Datelor Folosind Python
## Anul universitar 2025-2026

Organizarea din acest semestru este diferită.

Vom pune accent pe _prezentarea_ datelor, nu (doar) pe procesarea lor.

Se pot folosi unelte de inteligență artificială pentru procesare și programare, dar studenții trebuie să aibă capacitatea de a prezenta datele potrivit. Să aleagă formatul cel mai bun pentru reprezentare grafică, să calculeze cei mai potriviți indici statistici ș.a.m.d.

Astfel, la finalul semestrului, toți studenții ar trebui să aibă o foaie Jupyter Notebook cu o analiză complexă, care să conțină:
* prezentarea problemei și motivația pentru ea;
* planul de lucru;
* implementare (Python), cu elemente de statistică și vizualizare;
* interpretarea rezultatelor;
* concluzii.

Temele alese pot fi din zona de cercetare sau de interes public, de tip _data journalism_.

Implementarea preferată va fi pe platforma [Google Colab](https://developers.google.com/colab).

## Prima sarcină (mockup)
### Vaccinul antigripal
Sezonul rece aduce mereu înmulțirea cazurilor de gripă, cu sau fără complicații.

Generați aleatoriu un set de date (CSV), din care să extrageți situația vaccinărilor la nivel național. De exemplu:
- formulați o ipoteză de lucru;
- setul poate să conțină numărul total de persoane nevaccinate / vaccinate pe oraș / județ;
- prezentați statistici la nivel de județ/țară;
- mai generați un set de date cu veniturile medii / procentul de persoane cu bacalaureat / studii superioare;
- corelați rezultatele obținute, statistic (computațional) și grafic;
- prezentați concluziile.
